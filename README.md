# SEMOSAN (세모산)

> 실시간 밀착 등산 도우미 — 스마트 페이스메이커 모바일 앱

## 프로젝트 구조

```
semosan-mvp/
├── wireframes/     # Lo-fi 와이어프레임 (HTML)
└── (앱 코드는 추후 추가)
```

## 와이어프레임 보기

```bash
cd wireframes
python3 -m http.server 8080
# → http://localhost:8080
```
